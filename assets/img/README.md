## Image Guidelines

- **Format**: Use WebP for best performance, with JPG fallback
- **Size**: Optimize all images (use tools like TinyPNG, ImageOptim)
- **Dimensions**:
  - Project previews: 800x600px (4:3 aspect ratio)
  - Hero images: 1920x1080px (16:9 aspect ratio)
  - Team photos: 300x300px (square, 1:1 aspect ratio)
- **Quality**: 80-85% JPEG quality is optimal for web
